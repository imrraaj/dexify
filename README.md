# DEXify

A zero-fee Uniswap-powered trading terminal for swaps, token sends, chain switching, and TP/SL keeper orders.

## Highlights

- Multichain swap UI for supported Uniswap router networks
- Popular token lists per chain
- Wallet network switching
- Send tokens directly from the app
- TP/SL keeper order interface where supported
- Light and dark themes

> DEXify does not collect platform fees. Users still pay normal network gas and pool/router costs.
